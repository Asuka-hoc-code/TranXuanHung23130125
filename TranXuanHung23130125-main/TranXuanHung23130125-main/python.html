Học Python - Hướng dẫn lập trình từ cơ bản đến nâng cao
Giới Thiệu Cú Pháp Ví Dụ Kiểu Dữ Liệu Hàm Thư Viện Xử lý Ngoại Lệ

Giới Thiệu

Python là một ngôn ngữ lập trình bậc cao, dễ học và mạnh mẽ. Nó được sử dụng rộng rãi trong phát triển web, phân tích dữ liệu, trí tuệ nhân tạo, và nhiều lĩnh vực khác.

Cú Pháp Cơ Bản

Cú pháp của Python rất đơn giản và dễ hiểu. Dưới đây là một ví dụ về cú pháp cơ bản của một chương trình Python: 

print("Xin chào, Thế giới!")

Ví Dụ

Dưới đây là một ví dụ thực tế về việc sử dụng Python để tính tổng hai số: 

num1 = float(input("Nhập số thứ nhất: "))
num2 = float(input("Nhập số thứ hai: "))
sum = num1 + num2
print("Tổng:", sum)

Kiểu Dữ Liệu

Python hỗ trợ nhiều kiểu dữ liệu khác nhau, bao gồm:

  • Integer: Số nguyên
  • Float: Số thực
  • String: Chuỗi
  • Boolean: Đúng/Sai

Hàm

Hàm là một khối mã thực hiện một tác vụ cụ thể. Dưới đây là cách định nghĩa một hàm: 

def greet(name):
    return "Xin chào, " + name

print(greet("Thế giới"))

Thư Viện

Python có một bộ thư viện phong phú. Bạn có thể nhập thư viện bằng cách sử dụng từ khóa import. 

import math

print(math.sqrt(16))

Xử lý Ngoại Lệ

Xử lý ngoại lệ là cách để xử lý lỗi trong chương trình. Dưới đây là ví dụ về cách sử dụng try và except: 

try:
    x = 1 / 0
except ZeroDivisionError:
    print("Lỗi: Không thể chia cho 0")
