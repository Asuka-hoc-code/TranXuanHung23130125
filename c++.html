Học C++ - Hướng dẫn lập trình từ cơ bản đến nâng cao
Giới Thiệu Cú Pháp Ví Dụ Kiểu Dữ Liệu Hàm Lớp và Đối Tượng Xử lý Ngoại Lệ

Giới Thiệu

C++ là một ngôn ngữ lập trình bậc cao được phát triển từ ngôn ngữ C. Nó hỗ trợ lập trình hướng đối tượng và được sử dụng rộng rãi trong phát triển phần mềm, game, và ứng dụng hệ thống.

Cú Pháp Cơ Bản

Cú pháp của C++ rất gần với C. Dưới đây là một ví dụ về cú pháp cơ bản của một chương trình C++: 

#include <iostream>
using namespace std;

int main() {
    cout << "Xin chào, Thế giới!" << endl;
    return 0;
}

Ví Dụ

Dưới đây là một ví dụ thực tế về việc sử dụng C++ để tính tổng hai số: 

#include <iostream>
using namespace std;

int main() {
    int num1, num2, sum;
    cout << "Nhập số thứ nhất: ";
    cin >> num1;
    cout << "Nhập số thứ hai: ";
    cin >> num2;
    sum = num1 + num2;
    cout << "Tổng: " << sum << endl;
    return 0;
}

Kiểu Dữ Liệu

C++ hỗ trợ nhiều kiểu dữ liệu khác nhau, bao gồm:

  • int: Số nguyên
  • float: Số thực
  • char: Ký tự
  • string: Chuỗi

Hàm

Hàm trong C++ là một khối mã thực hiện một tác vụ cụ thể. Dưới đây là cách định nghĩa một hàm: 

#include <iostream>
using namespace std;

void greet() {
    cout << "Xin chào, Thế giới!" << endl;
}

int main() {
    greet();
    return 0;
}

Lớp và Đối Tượng

C++ hỗ trợ lập trình hướng đối tượng. Dưới đây là ví dụ về lớp và đối tượng: 

#include <iostream>
using namespace std;

class Dog {
public:
    void bark() {
        cout << "Gâu gâu!" << endl;
    }
};

int main() {
    Dog myDog;
    myDog.bark();
    return 0;
}

Xử lý Ngoại Lệ

C++ cho phép xử lý ngoại lệ bằng cách sử dụng try và catch. Dưới đây là ví dụ: 

#include <iostream>
using namespace std;

int main() {
    try {
        throw 20; // ném một ngoại lệ
    } catch (int e) {
        cout << "Ngoại lệ: " << e << endl;
    }
    return 0;
}
